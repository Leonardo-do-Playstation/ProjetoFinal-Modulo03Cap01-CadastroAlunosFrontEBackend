## 📂 Estrutura do Projeto


/frontend → Interface do usuário (Angular)
/backend → API REST em Java (Spring Boot)


---

## ▶️ Como Executar o Projeto

### 🔹 Pré-requisitos

- Node.js instalado
- Angular CLI
- Java JDK
- Maven ou Gradle

---

### 🔹 Para rodar backend:
cd backend
./mvnw spring-boot:run
ou
mvn spring-boot:run

🔹 Para rodar frontend
cd frontend
npm install
ng serve

Acesse no navegador:

http://localhost:4200

🔗 Comunicação entre Frontend e Backend

O frontend consome a API REST do backend para realizar operações de CRUD (Create, Read, Update, Delete).
